### Desafio: Verificador de Maioridade\!

Seu programa vai pedir a idade do usuário e, em seguida, informar se ele é maior ou menor de idade, de acordo com as regras brasileiras (18 anos).

**O que você precisa fazer:**

1.  **Peça ao usuário para digitar sua idade.** (Lembre-se: use `input()` e **converta** para um número inteiro com `int()`).
2.  **Crie uma condição (`if`)** para verificar se a idade digitada é **maior ou igual a 18**.
3.  **Dentro do `if`** (com a **indentação** correta\!), exiba uma mensagem como: `"Você é maior de idade no Brasil!"`
4.  **Crie um `else`** para a situação em que a idade **não for maior ou igual a 18**.
5.  **Dentro do `else`** (também com a **indentação** correta\!), exiba uma mensagem como: `"Você é menor de idade no Brasil."`

-----

**Exemplo de como deve funcionar (com duas possibilidades de entrada):**

**Cenário 1: Usuário digita 20**

```
Qual é a sua idade? 20
Você é maior de idade no Brasil!
```

**Cenário 2: Usuário digita 15**

```
Qual é a sua idade? 15
Você é menor de idade no Brasil.
```
